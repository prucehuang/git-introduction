# Markdown编辑
Markdown语法 | 效果

输出后的效果 | 快捷键
--|--
Bold | **text**	Ctrl/⌘ + B
Emphasize	*text*	Ctrl/⌘ + I
Strike-through	~~text~~	Ctrl + Alt + U
Link	[title](http://)	Ctrl/⌘ + K
Inline Code	`code`	Ctrl/⌘ + Shift + K
Image	![alt](http://)	Ctrl/⌘ + Shift + I
List	* item	Ctrl + L
Blockquote	> quote	Ctrl + Q
H1	# Heading	
H2	## Heading	Ctrl/⌘ + H
H3	### Heading
